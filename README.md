# Niflect 示例: HelloWorld

**NiflectSampleHelloWorld** 是 Niflect 的入门示例, 可通过该示例体验使用者视角下 Niflect 的主要特性:

- 非侵入式
  - 不要求继承自某个类, 这意味着使用者甚至能为三方库类型生成反射元数据
  - 运行时中的反射宏标签为空
- 任意类型反射与序列化
  - 如嵌套模板, 任意指针等类型
  - 通过不依赖任何格式的 RwNode 保存载入实例
